Runs the exact same pipeline live — Stage-0 shortlist → BM25 → full scorer — and returns a downloadable top-100 CSV. No file handy? The Top 100 tab already shows the full 100,000-candidate ranking.

Two-phase pipeline

Phase A precomputes 29 features per candidate and a BM25 index offline. Phase B — the timed step — filters 100K → 2,000 (Stage 0), then deep-scores the shortlist to a Top 100 in ~28 seconds.

Six additive base scores

  • Skill Trust — 34%
  • Career Trajectory — 20%
  • Experience Fit — 16%
  • Lexical (BM25) — 14%
  • Location — 10%
  • Education — 6%

Three multiplicative gates

Role-Fit crushes keyword stuffers (a reject-title profile is multiplied by 0.15). Integrity zeroes out fraudulent profiles — it eliminates exactly 69 honeypots in the 100K pool. Availability scales by responsiveness & activity.

Fact-grounded reasoning

Every reasoning string traces to a real number in the candidate's profile — title, years, matched skill groups, concerns. No hallucinated employers or skills.

Deterministic · rule-based · reproducible. Same code path as rank.py. 34 automated tests.
